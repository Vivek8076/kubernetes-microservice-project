# Kubernetes Microservice Project 🚀

## Tech Stack
- Node.js
- MongoDB
- Docker
- Kubernetes

## Features
- Deployment & Service
- Ingress (domain-based access)
- ConfigMap & Secret
- Persistent Volume (MongoDB)
- HPA (Auto Scaling)

## How to Run
1. Build Docker image
2. Push to DockerHub
3. Apply Kubernetes YAML files
